## Actividad Guiada 4 – Java

Este proyecto contiene 3 ejercicios desarrollados en Java:
- Calculadora
- Ordenamiento Burbuja
- Consumo de API

La carpeta principal se llama **Actividad Guiada 4**. Dentro de ella se encuentran subcarpetas para cada ejercicio.

## Ejecución de los ejercicios

Ejecutar los siguientes comandos desde la carpeta **Actividad Guiada 4**:

### **Calculadora**
- **Compilar:** `javac Calculadora/Calculadora.java`
- **Ejecutar:** `java Calculadora.Calculadora`

### **Ordenamiento Burbuja**
- **Compilar:** `javac Burbuja/Burbuja.java`
- **Ejecutar:** `java Burbuja.Burbuja`

### **Consumo de API**
- **Compilar:** `javac Api/ApiExample.java`
- **Ejecutar:** `java Api.ApiExample`

## Nota

Los comandos pueden variar ligeramente en Windows (por ejemplo usando `\` en lugar de `/`), pero la lógica es la misma.
